Web Workers Demo

Have you ever wonder what is the result of 1 + 2 + 3 + 4 + ... + 2,500,000,000? If so, I have the solution to your question.

This is a quite simple example of how we can delegate a heavy process to a new thread instead of completely blocking the UI.

Standard "For"

With a standard For your browser will be frozen cause the calculation keeps it so busy that cannot respond to UI interactions, not even a progress bar is displayed (the code to show it is there, thrust me!). Once the calculation is done, the browser goes back to normal operation.

Web Worker

With a web worker running on the background, your browser is happy and can show a progress bar while the calculation is in progress.
